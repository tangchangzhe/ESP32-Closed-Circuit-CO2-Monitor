# 硬件接线说明

## 组件清单

| 组件         | 型号               | 数量 | 备注               |
| ------------ | ------------------ | ---- | ------------------ |
| 主控板       | ESP32-S3-DevKitC-1 | 1    | 3.3V IO            |
| CO2传感器    | ZG09SR             | 1    | NDIR红外，UART接口 |
| 气泵         | 5V微型气泵         | 1    | 用于气路采样       |
| 过滤瓶       | -                  | 1    | 干燥管/除湿        |
| 密封盒       | 气体超密封盒       | 1    | 放置被测样品       |
| 微型气室     | 铝箔材质           | 1    | 包裹传感器         |
| 反倒吸装置   | -                  | 1    | 防止气体回流       |

## ZG09SR 传感器

ZG09SR 是深圳盛世物联生产的 NDIR (非色散红外) CO2 传感器：

- 测量范围: 0-5000ppm (可选更高量程)
- 精度: ±(50ppm + 5%读数)
- 接口: UART (9600bps, 8N1)
- 工作电压: 4.5-5.5V
- UART 信号兼容 3.3V

## 电源说明

```
USB 5V ──┬── ESP32-S3 (内部3.3V稳压)
         │
         ├── ZG09SR (5V)
         │
         └── 气泵 (5V)
```

- ESP32-S3：通过 USB-C 供电，内部稳压提供 3.3V IO
- ZG09SR：需要 5V 供电，UART 信号 3.3V 兼容，可直连 ESP32
- 气泵：5V 供电，可通过 GPIO 控制继电器/MOS管开关

其他供电方案：
- 外接 5V 电源适配器
- 18650 锚电池 + 升压模块

## 引脚连接

### ZG09SR CO2 传感器 (UART)

| ESP32-S3 | ZG09SR | 说明       |
| -------- | ------ | ---------- |
| GPIO 16  | TXD    | 传感器发送 |
| GPIO 17  | RXD    | 传感器接收 |
| 5V       | VCC    | 电源       |
| GND      | GND    | 地         |

## 接线示意

```
                    ESP32-S3-DevKitC-1
                   ┌─────────────────┐
                   │      USB-C      │
                   │    (5V 供电)    │
                   │                 │
   ZG09SR TXD ───► │ GPIO16 (RX)     │
   ZG09SR RXD ◄─── │ GPIO17 (TX)     │
                   │                 │
              5V ──│ 5V          GND │── GND
                   │                 │
                   └─────────────────┘
         (ZG09SR 和气泵共用 5V)
```

## 气路设计

本项目采用的气路布置：

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  进气口 → 过滤瓶 → [密封盒] → 气泵 → [气室+传感器] → 反倒吸 → 出气 │
│           (干燥)    (样品)           (铝箔微型)      (防回流)     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 各部件功能

| 部件             | 功能                                   |
| ---------------- | -------------------------------------- |
| 过滤瓶（干燥管） | 除湿，防止水汽影响传感器               |
| 气体超密封盒     | 放置被测样品（如藻类培养物）           |
| 气泵             | 抽取密封盒内气体                       |
| 铝箔微型气室     | 包裹传感器，形成小体积检测腔           |
| 反倒吸装置       | 防止关闭气泵时气体回流污染样品         |

### 设计说明

- 气泵位于密封盒之后、传感器之前，形成**负压抽取**
- 铝箔气室缩小检测体积，提高响应速度
- 反倒吸装置确保气路单向流动

### 关于漏气

即使使用“超密封盒”，长时间运行仍可能存在微小漏气，导致 CO2 数据持续下降。这正是本项目开发**漂移补偿算法**的原因——通过软件修正弥补硬件密封的不足。

## 自定义引脚

如需修改引脚分配，编辑 `firmware/src/config.h`:

```cpp
// ZG09SR UART
#define CO2_RX_PIN    16  // 接传感器TXD
#define CO2_TX_PIN    17  // 接传感器RXD
```
